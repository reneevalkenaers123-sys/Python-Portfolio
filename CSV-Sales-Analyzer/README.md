# CSV Sales Analyzer

Interactive Python project that analyzes sales data from a CSV file.

## Features

- View total sales
- Show sports category products
- Find best selling product
- Menu based user interaction

## Technologies Used

- Python
- CSV module
- Loops
- If statements
- Functions / logic

## Files

- sales_analyzer.py → main program
- sales_data.csv → dataset

## Goal

Practice Python fundamentals with real data.
